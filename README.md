# 项目简介
## 核心功能
- 用户认证模块：登录验证，权限控制
- 学生信息管理模块：基本信息管理，学籍管理，毕业管理
- 课程与成绩管理模块：课程安排，成绩录入，成绩查询
- 奖惩情况管理模块：奖励记录，惩罚记录
- 系统管理模块：用户管理，角色管理
## 适用场景
- 大学生信息管理
- 教学教务管理
- 成绩与奖惩管理
## 技术特点
- 基于Python开发，具有良好的可扩展性和稳定性
- 采用前后端分离架构，便于维护和升级
- 数据库结构设计合理，确保数据安全性和高效查询
- 系统功能全面，覆盖学生管理的各个方面
